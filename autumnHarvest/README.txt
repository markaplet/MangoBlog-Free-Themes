INSTALLATION NOTES
-------------------------------------------------------------------
The Autumn Harvest theme uses jQuery for the menu effect. Included with this theme is the minified version of jQuery (v1.3.2) If your including other jQuery scripts on your page such as Seb Duggans "jQuery plugin for Mango" http://sebduggan.com/posts/jquery-plugin-for-mango, be sure to remove the jQuery call on line 25 of the meta.cfm file. 

The meta.cfm contains all the links to css and javascript files as well as other meta related content like site description, keywords, and site favicon.
